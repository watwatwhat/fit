FitTrack プライバシーポリシー

最終更新日: 2024年12月19日
重要: watwatlab（以下「当社」）は、お客様のプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーは、フィットネス・健康管理アプリケーション「FitTrack」（以下「本アプリ」）における個人情報の取り扱いについて説明します。

1. 収集する情報

1.1 お客様が提供する情報

情報の種類 具体的な内容 収集目的
アカウント情報 メールアドレス、ユーザー名、プロフィール写真 アカウント管理、認証
基本情報 身長、体重、年齢、性別 健康管理機能の提供、カロリー計算
健康データ 体重記録、体脂肪率、BMI、血圧、心拍数 健康状態の追跡、分析
運動データ ワークアウト記録、歩数、消費カロリー、運動時間 フィットネス目標の管理、進捗追跡
栄養データ 食事記録、摂取カロリー、栄養素 栄養管理、健康目標の達成支援
睡眠データ 睡眠時間、睡眠の質 睡眠パターンの分析、健康管理

1.2 自動的に収集される情報

  • デバイス情報: デバイスの種類、OS、アプリのバージョン
  • 利用状況: アプリの使用頻度、機能の利用状況
  • 技術情報: IPアドレス、ログ情報、クラッシュレポート

1.3 外部サービスからの情報

HealthKit/Google Fit連携:
お客様の同意に基づき、以下の健康データを取得する場合があります：
  • 歩数、距離、消費カロリー
  • 心拍数、血圧
  • 睡眠データ
  • 体重、体脂肪率
  • ワークアウト記録

2. 情報の利用目的

収集した個人情報は、以下の目的で利用します：

  1. サービス提供: 健康管理、フィットネス追跡機能の提供
  2. パーソナライゼーション: お客様に合わせた健康アドバイス、目標設定
  3. 分析・改善: アプリの機能改善、新機能開発
  4. 通知: リマインダー、健康に関するアドバイスの送信
  5. サポート: お客様からのお問い合わせ対応
  6. セキュリティ: 不正利用の防止、セキュリティ向上
  7. 法的義務: 法令に基づく対応

3. 情報の共有

当社は、以下の場合を除き、お客様の個人情報を第三者と共有することはありません：

3.1 お客様の同意がある場合

  • 明示的な同意をいただいた場合
  • ソーシャル機能での情報共有（お客様が選択した場合のみ）

3.2 サービス提供のための共有

  • クラウドサービス: Firebase（Google）- データ保存、認証
  • 分析サービス: Firebase Analytics - 利用状況分析
  • クラッシュ分析: Firebase Crashlytics - アプリの安定性向上

3.3 法的要求がある場合

  • 法令に基づく開示要求
  • 裁判所命令
  • 緊急時の生命・身体の保護

4. データの保存と保護

4.1 データの保存場所

  • お客様のデバイス（ローカルストレージ）
  • Firebase（Google Cloud Platform）- 暗号化されたクラウドストレージ

4.2 セキュリティ対策

  • 暗号化: 保存時および転送時の暗号化
  • アクセス制御: 最小権限の原則に基づくアクセス管理
  • 認証: Firebase Authenticationによる安全な認証
  • 監視: 不正アクセスの監視と検知

4.3 データの保存期間

  • アカウント情報: アカウント削除まで
  • 健康データ: お客様が削除するまで、または法的要求に基づく期間
  • ログデータ: 最大2年間

5. お客様の権利

お客様は、ご自身の個人情報について以下の権利を有します：

5.1 アクセス権

  • 保存されている個人情報の確認
  • データのエクスポート機能

5.2 修正権

  • アプリ内での情報の編集・更新
  • 不正確な情報の修正要求

5.3 削除権

  • アカウントの削除
  • 特定のデータの削除

5.4 制限権

  • データ処理の制限要求
  • 通知設定の変更

6. 第三者サービスとの連携

6.1 HealthKit（iOS）

  • Appleの健康アプリとの連携
  • お客様の明示的な許可が必要
  • データはAppleのプライバシーポリシーにも準拠

6.2 Google Fit（Android）

  • Googleの健康プラットフォームとの連携
  • お客様の明示的な許可が必要
  • データはGoogleのプライバシーポリシーにも準拠

6.3 Firebase Services

  • 認証、データベース、分析機能
  • Googleのプライバシーポリシーが適用
  • GDPR、CCPA準拠

7. 国際的なデータ転送

お客様の個人情報は、サービス提供のため以下の地域に転送される場合があります：

  • アメリカ合衆国: Firebase/Google Cloud Platform
  • 適切な保護措置: 標準契約条項、認定された転送メカニズム

8. 未成年者のプライバシー

  • 13歳未満のお客様からは意図的に個人情報を収集しません
  • 13歳未満の利用が判明した場合、速やかにデータを削除します
  • 保護者の方は、お子様の利用状況を監督してください

9. Cookieとトラッキング技術

9.1 使用する技術

  • ローカルストレージ: アプリの設定、ユーザー設定の保存
  • 分析SDK: Firebase Analytics - 利用状況の分析
  • クラッシュレポート: Firebase Crashlytics - エラー分析

9.2 オプトアウト

  • アプリ設定から分析データの送信を無効化可能
  • デバイス設定から広告IDの制限が可能

10. プライバシーポリシーの変更

  1. 本プライバシーポリシーは、法令の変更やサービスの改善に伴い更新される場合があります
  2. 重要な変更がある場合は、アプリ内通知またはメールでお知らせします
  3. 変更後も継続してサービスをご利用いただくことで、変更に同意したものとみなします
